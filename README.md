# docker-php

Пробую собрать удобную среду для PHP разработки.  
Основные требования:

- Минимализм: ничего лишнего, только необходимое.
- Простая и понятная конфигурация
- Минимально возможный размер
- Необходимые контейнеры:
  - Apache, или Nginx + PHP FPM (через unix socket).  
    Сборка в 1-м контейнере на основе Alpine Linux.
  - MySQL, или MariaDB
  - Mailpit - для тестирования Email и SMTP

## Синхронизация UID/GID с хостом
По умолчанию конфигурация настроена на UID/GID 1000/1000.  
Если на вашей хост системе эти значения отличаются от 1000,
то переименуйте .env.dist в .env и впишите туда актуальные значения.
Проверить можно так (на хост системе): `id -u && id -g`
